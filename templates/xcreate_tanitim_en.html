Technical Details

Solid Foundation

🛡️

Güvenlik & Doğrulama

All SQL queries are protected. File uploads validate extension and file size. Conditional field validation — hidden fields are not validated.

SQL injection protection File type validation XOOPS security integration
⚡

Performance

Query results are converted to PHP arrays and reused. Helper methods cache table existence. If the rating table is missing it's created automatically — the page never breaks.

Query cache Lazy table creation Output buffer management

6 optimized tables. Slugs guaranteed fast via unique index. Rating system in a separate table for a modular structure.

xcreate_categories xcreate_items xcreate_fields xcreate_field_options xcreate_field_values xcreate_ratings

XOOPS Integration

Full ORM support via XoopsPersistableObjectHandler. XOOPS Smarty3-compatible plugin system. Breadcrumb, pagination and user system integration.

Smarty3 Plugin

function.xcreate.php

Call xcreate content with a single tag from any Smarty template. Works outside module pages too.

🧩 Smarty
Plugin

xcreate content directly from theme templates

In any (.tpl) XOOPS theme file, {xcreate} tagını kullanarak tag to list xcreate content. Filter by category, set custom ordering, load custom fields and specify your own template.

Works in theme templates cat_id or cat_slug filter Custom field support 4 sort modes Debug mode
⚡

Basic Usage

Show the latest 10 items using your own template.

{xcreate template="game_list"}
📂

Categories Filtresi

Filter by a specific category using its ID or slug.

{xcreate
  cat_slug="news"
  template="news_card"
  limit="5"}
🔀

Sorting & Pagination

Sort randomly, by date, title or hit count.

{xcreate
  order="random"
  limit="6"
  from="0"}
🐛

Debug Modu

See parameters, template paths and field counts in HTML comments.

{xcreate
  template="test"
  debug="1"}

⚙️ All Parameters

ParametreTipVarsayılanAçıklama
templatestring"xcreate_custom"Template name to use (without .tpl extension). Theme folder is searched first, then xcreate/templates/.
categoryint0Filter by category ID. 0 = all categories.
cat_slugstring""Categories slug ile filtrele. category parametresine göre önceliklidir.
limitint10How many items to retrieve.
fromint0Start offset. Used for pagination.
orderstring"date"Sort by: date · title · hits · random
sortstring"DESC"Sort direction: DESC veya ASC
statusint11 = published, 0 = draft, 2 = pending.
fieldsint11 = load custom fields. 0 = disabled (for performance).
debugbool—Active when parameter is present. Debug comments appear in HTML source.
